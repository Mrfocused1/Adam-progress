# ADAM PROGRESS

Premium one-page portfolio site for combat sports athlete and creator **Adam "The Berserker" Progress** (Tibaba Progress) — 6'8" light-heavyweight out of Hamburg, Germany.

Cinematic dark-and-red brand aesthetic — built for attracting sponsors and partnerships.

## Stack

- Plain HTML, Tailwind CSS (CDN), vanilla JS
- Google Fonts: Anton, Bebas Neue, Inter, Montserrat, Permanent Marker
- Self-hosted DBX 6 full fight (H.264 MP4) + YouTube embeds for other fights

## Run locally

```bash
python3 -m http.server 8787
# open http://localhost:8787/
```

## Sections

Hero · Identity strip · About · Stats · Fights (tabs: Full Fights / Shorts, click-to-play inline) · Social proof · Content pillars · Why partner · Final CTA
